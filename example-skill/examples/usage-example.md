# 스킬 사용 예시

## Claude에게 요청하는 방법

### 예시 1: API 엔드포인트 생성
```
"backend-api-helper 스킬을 사용하여 사용자 정보를 조회하는 GET API 엔드포인트를 만들어줘.
요구사항:
- URL: /api/v1/users/:id
- 인증 필요
- 사용자 정보 반환"
```

### 예시 2: 에러 핸들링 개선
```
"backend-api-helper 스킬을 참고하여 현재 API의 에러 핸들링을 개선해줘.
현재 코드에서 에러 응답 형식이 일관되지 않아."
```

### 예시 3: API 설계 리뷰
```
"backend-api-helper 스킬의 가이드라인을 따라서 내가 만든 API 설계를 리뷰해줘:
[여기에 API 설계 내용 붙여넣기]"
```

## 예상되는 결과

Claude는 이 스킬의 지침을 따라서:
- RESTful 원칙을 따르는 API 엔드포인트 생성
- 일관된 에러 핸들링 구현
- 적절한 HTTP 상태 코드 사용
- 보안을 고려한 코드 작성
